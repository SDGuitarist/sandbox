Fix: add `get_all_templates` to the existing wizard wiring entry for `template_models.py`.

But the Export Names Table lists only "wizard" as a consumer, contradicting the wiring table and the seed code. Fix: add "seed" to the "Used By" column for `create_prompt` in the Export Names Table.

But the Export Names Table lists only "grading" as a consumer, contradicting the wiring table and the seed code. Fix: add "seed" to the "Used By" column for `save_grade` in the Export Names Table.
