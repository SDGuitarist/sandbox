# Demo Spec (worktree-base / STALE copy)

This stands in for the spec AS IT EXISTS AT THE WORKTREE BASE (the repo's default
branch, where worker worktrees root). It is the STALE version — analogous to the
2010-line spec all 16 workers read in Run 070 while the gates had validated the
2295-line converged spec on the feature branch.

The runner commits this on the default branch. Its only job is to differ from the
gated copy, so the FC52 detector reports PROVENANCE_DRIFT.
